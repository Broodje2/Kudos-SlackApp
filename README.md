# Kudos slack bot

A bot to add some friendly rivalry to your team

## Overview

This is a Slack app built with the [Bolt for JavaScript framework][2] and openAi integrated to have smart and dynamic responses. The main purpose 
is to add a friendly game to your work environment where people can award eachother with points for anything. The app has a built in economy and people have to battle for their spot on the leaderboard.

## Functionality

- An award system based on teammates
- AI responses based on messages sent by anyone that show grattitude
- A leaderboard
- A shop with custom items

## Project purpose

The purpose of this project was to make a interesting way to use the flipdotboard in the office of OWOW.

## Tools that were used

- Bolt for JavaScript framework
- OpenAI

## Installing

First you have to clone the repository in your code editor, preferably Visual Studio Code

```bash
git clone https://github.com/Broodje2/Kudos-SlackApp.git
```

Then go in the folder you downloaded

```bash
cd /Kudos-SlackApp
```

### And lastly install all dependencies

```bash
npm install
```

## Configuration

To get all the funcionality you have to make a .env file and ask Niels for all the needed tokens.

## Running locally

To run the bot locally, run the following command:

```bash
npm run start
```
